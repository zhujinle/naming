# 点名器

英语老师让我写一个随机点名的软件，于是乎...

## 本项目优点

快速部署，一个fork后利用GithubPage进行**在线使用**

## 如何使用

- 在线使用

1. 点击本页面右上角fork即可

- 离线使用

1. 下载本项目

```bash
# 下载项目
git clone git@github.com:Himself65/pointer.git 
```

或者下载压缩包也可以

2. 安装所需库

```bash
gem install jekyll
```

3. 运行

```bash
cd pointer

jekyll serve
```
